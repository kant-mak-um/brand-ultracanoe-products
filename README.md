# brand-ultracanoe-products

**Brand:** UltraCanoe  
**Domain:** https://ultracanoe.com (apex) and https://www.ultracanoe.com  
**Category:** Modular canoes — outrigger, whitewater, packable, and standard

## Purpose
This repository hosts the public-facing product definitions, documentation, and marketing-ready assets for **UltraCanoe**.

UltraCanoe focuses on **modular canoe systems** that can be configured for multiple sub-markets:
- packable / transport-friendly builds
- whitewater-capable configurations
- outrigger-enabled stability variants
- standard touring and utility canoes

## Commerce & Memberships
Sales, memberships, and checkout will be handled by **WaterWolf** (private platform).  
This repo is intentionally **public** and contains only publishable product and documentation assets.

## What’s in this repo (initial)
- `/site/` – static website content (landing pages and product pages)
- `/catalog/` – product placeholders (to be populated as kits and accessories are finalized)
- `/LICENSE` and `/LICENSES/` – licensing scaffold (per your standard)

## Status
Pre-launch. Content will expand as production ramps and UltraCanoe kits/accessories are released.
